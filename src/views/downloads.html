System Requirements

Zibbit is a Microsoft Word Add-in with the following system requirements:

  • Windows Operating System
  • Microsoft Word (2007 or later)
  • Microsoft .NET Framework (4.7.2 or later)
  • Microsoft Visual Studio 2010 Tools for Office Runtime (10.0.30319 or later)
  • Microsoft Edge WebView2 Runtime (Learn more or Download)
